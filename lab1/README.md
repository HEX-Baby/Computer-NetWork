```
│
└── x64/
    └── Debug/

```
  
# 使用方法
  
先双击打开 `ChatServer.exe` ，然后双击打开 `ChatClient.exe` ，`ChatClient.exe`想打开几个就打开几个。
  
然后在 `ChatClient.exe` 中输入名称，之后就可以开始聊天了。输入`exit`退出。
